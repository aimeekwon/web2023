Webpack

React

Webpack은 모듈 번들러(module bundler)로서 JavaScript 파일 및 다른 웹 자원(이미지, CSS 등)을 번들로 묶는 도구입니다.

웹팩을 사용하면 여러 개의 모듈로 구성된 프론트엔드 애플리케이션을 하나의 파일로 묶어줄 수 있습니다. 웹팩은 주로 웹 개발에서 사용되며, 모듈 시스템(CommonJS, AMD, ES 모듈)을 사용하는 자바스크립트 파일들을 의존성 그래프로 만들고, 이를 기반으로 번들링하여 정적 파일을 생성합니다. 이렇게 번들된 파일은 웹 애플리케이션에서 로드되어 사용될 수 있습니다.

2. JSX

JSX(JavaScript XML)는 React에서 UI를 구성하는 데 사용되는 문법입니다. JSX는 HTML과 매우 유사하게 생겼지만, JavaScript를 확장한 문법입니다. JSX는 React 컴포넌트에서 UI를 정의할 때 사용되며, 이를 사용하면 JavaScript 코드 안에 HTML 코드를 작성할 수 있습니다.
JSX는 React에서 UI를 작성하는 데 매우 편리한 문법이며, JavaScript 코드와 HTML 코드를 혼합하여 작성하므로 가독성이 높고, 개발자들이 UI를 더 쉽게 작성할 수 있도록 도와줍니다.

import React from "react";
import ReactDOM from "react-dom/client";
const name = "aimee";
const hello = <h1>hello {name}</h1>;
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);
// hello aimee

객체와 함수를 같이 사용한 경우

import React from "react";
import ReactDOM from "react-dom/client";
function helloName(){
    return name.nick;
};
const name = {
    nick : "aimee"
}
const hello = <h1>Hello, {helloName()}</h1>;
const root = ReactDOM.createRoot(document.getElementById('root'));
root.render(hello);
// Hello, aimee
